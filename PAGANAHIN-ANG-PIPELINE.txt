╔══════════════════════════════════════════════════════════════╗
║  TROUBLESHOOTING                                            ║
╚══════════════════════════════════════════════════════════════╝

Problem: "Python was not found"
Solution: I-install ang Python muna
          https://www.python.org/downloads/

Problem: "Database connection failed"
Solution: Check kung running ang MySQL
          Services → MySQL → Start

Problem: Walang deals sa pipeline
Solution: Ulitin ang Step 1 (ADD-SAMPLE-DATA.bat)

Problem: Backend error
Solution: Check backend/.env kung tama ang DB password
